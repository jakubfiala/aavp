ripple is a simple, yet powerful interface for live MIDI sequencing, written in C++/openframeworks. Inspired by instruments such as the tenori-on, it consists of a grid of buttons that correspond to individual MIDI pitch values. The user first creates a sequence by selecting a button, and then toggles the playback mode, where by clicking on a particular button, a tempo-synced ‘wave’ is created. Whenever the wave encounters a toggled button, a note is triggered polyphonically. The app can be used as live MIDI input into synths in any kind of DAW — however, it is necessary to create a virtual MIDI routing, e.g. with midi-patchbay.
maxiBark is an implementation of Bark scale spectral analysis in Maximilian, a C++ DSP library by Mick Grierson/Chris Kiefer. It works with the ofxMaxim plugin for openframeworks. It has three methods for obtaining loudness based on this perceptual scale:
  • specificLoudness – an array of absolute loudness values for each of the 24 Bark bands
  • relativeLoudness – loudness of each Bark band, relative to the loudest band
  • totalLoudness – absolute loudness of all bands, summed
